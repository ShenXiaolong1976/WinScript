.NET Decompilation
============================================
!mex.codescope             Prints all available code analysis checklists
!mex.decompilemember       Decompile and print psuedo-C# source code for the given [MemberName]
!mex.decompiletype         Decompile and print psuedo-C# source code for the given [TypeName]
!mex.il                    Prints the IL for the specified method
!mex.printmanifest         Prints the assembly manifest for the specified module
!mex.printmembers          Scans specified module and type [Module!TypeName] and prints all members
!mex.printtypes            Scans specified [Module] and prints all types
!mex.runcheck (!runchecks) runs the specified check(s) on the specified module(s)
!mex.runchecklist          runs the specified checklist(s) on the specified module(s)
!mex.spdisposecheck        Executes the SharePoint Dispose and Do Not Dispose Checklist items
!mex.whocalls              Scans all loaded managed modules and finds methods that call [MethodName]
!mex.whoimplements         Scans all loaded managed modules and finds types that implement [InterfaceName]
!mex.whoinherits           Scans all loaded managed modules and finds types that inherit [TypeName]
!mex.whonews               Scans all loaded managed modules and finds methods that construct [TypeName]
!mex.whopins               Scans managed modules and all finds methods that pin objects of a given [TypeName] or all types
